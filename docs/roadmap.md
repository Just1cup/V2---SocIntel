# Roadmap

- Establish repo structure
- Stand up API, worker, and web shell
- Define domain models and API contracts

## Phase 2

- Add authentication and user management
- Persist cases and investigations
- Create analysis job lifecycle

## Phase 3

- Port legacy IOC analyzers
- Add provider adapters and job workers
- Introduce caching and rate limiting

## Phase 4

- Add teams, permissions, and audit logs
- Harden observability and deployment
